# Gerar CSV agrupado do arquivo 1802002.txt

Este repositório inclui o script `scripts/gerar_csv_1802002.py`, que lê o arquivo `1802002.txt` (formato “dump” textual) e:

- identifica funcionários por blocos de 4 linhas consecutivas:
  1. `matricula:  "..."`  
  2. `nome:  "..."`  
  3. `cpf:  "..."`  
  4. `valor:  <numero>`
- soma o campo `valor` por `matricula`
- grava `1802002_agrupado.csv` **ordenado por matrícula**

## Requisitos

- Python 3.8+ (sem dependências externas)

## Como executar

Na raiz do repo:

```bash
python3 scripts/gerar_csv_1802002.py
